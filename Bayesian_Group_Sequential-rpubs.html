Adaptive Design

  • What is adaptive design?

    • pre-specifed changes can be made based on analyses of accumulating data whilst maintaining the validity and integrity of the trial

Adaptive Design

  • Why we want to use adaptive design?

    • flexibility \( \Rightarrow \) efficient trials

  • When to use adaptive design?

    • used in Phase I - Phase III

Adaptive Design

  • What adaptive design can do?

    • abandoning treatments or doses ( CRM, E-WOC, multi-arm multi-stage, adaptive dose-ranging )
    • changing the allocation ratio of patients to trail arms ( adpative randomization)
    • rapid transitioning between phases ( seamless phase I/II, seamless phase II/III )
    • stopping the whole trial at an early stage for success or lack of efficacy ( group sequential )
    • refinning the sample size ( sample size re-estimation )
    • identifying patients most likely to benefit and focusing recruitment efforts on them ( population enrichment )
